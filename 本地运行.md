# 本地运行
  ## 创建虚拟环境
  ```shell
    python -m venv venv
    # macOS and Linux 激活虚拟环境
    source venv/bin/activate
  ```
  ## 安装 python 依赖
  ```shell
    pip install -r requirements.txt
  ```
  ## 运行项目
  ```shell
    python start.py
  ```
  运行成功后在浏览器中访问 http://0.0.0.0:80 即可
  API文档地址：http://0.0.0.0:80/docs
  新增的接口地址： /api/bilibili/web/down_video
  注意：在本地运行必须安装 ffmpeg 工具，要支持 ffmpeg 命令，否则会报错。

# 在 bilibili_web 中新增一个 down_video 方法，用于下载视频。
    方法类型：GET
    方法参数：
    - bv_id: 作品id（必填）
    - remove_watermark: 是否移除水印（可选，默认值为 0:不移除水印，1:移除水印）
    - resolution: 视频分辨率（可选，默认值为 0:原分辨率，1:720p，2:1080p）
    返回值：
    - 参考改页面的其它接口，返回下载成功即可
    - 下载的视频文件要保存到项目根目录下的 video 文件夹中，视频格式为 mp4，文件名格式为 bv_id.mp4
    - 如果视频文件已存在，则直接返回已存在的文件路径
    - 如果传入 remove_watermark 参数为 1，则下载的视频文件要移除水印
    特别说明：
    - 使用 ffmpeg 工具来处理视频文件，确保本地安装了 ffmpeg 工具。
    - 水印一般在右上角，并且包含 bilibili 字样。
    - 关键代码添加注释说明
